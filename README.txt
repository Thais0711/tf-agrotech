TF AgroTech - Cadastro por Fazenda

Inclui:
- Tela pública para cada fazenda criar sua própria conta
- Cada fazenda vê somente seus próprios dados
- Administrador da fazenda cria funcionários
- Login individual
- Recuperação de senha
- Gastos, saídas, rebanho, produção e relatórios
- Backup
- Relatório para salvar/imprimir em PDF

Como rodar:
pip install flask
python app.py
http://127.0.0.1:5000
